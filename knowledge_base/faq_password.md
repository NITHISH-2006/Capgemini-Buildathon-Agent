# Password Reset

## How do I reset my password?
If you forgot your password, you can use the "Forgot Password" link on the login page. Alternatively, you can ask our Intelligent Support Agent to reset your password. The agent will ask for your user_id and automatically issue a temporary password for you to log in. Please ensure you change this temporary password immediately after logging in.

## My temporary password expired, what should I do?
Temporary passwords are valid for 24 hours. If it has expired, please request a new password reset.
